# 星海量子通用操作系统 - 配置终端

**版本**：1.0.0  
**开发者**：蓝域星河有限公司集团  

星海量子通用操作系统是一个多用户、多语言、高安全性的量子操作系统配置与管理终端。  
它提供了系统初始化、用户管理、终端命令交互、虚拟文件系统、定时任务、升级维护、安全审计等完整功能，并支持 GitHub 远程更新。

---

## 快速开始

### 环境要求
- Python 3.8+
- 依赖库：`bcrypt`, `cryptography`, `requests`, `psutil`, `pyotp`, `paramiko`, `croniter`

### 安装与运行
```bash
cd 组件
pip install -r requirements.txt
python main.py
```
首次使用请先进行 系统初始化，按提示输入产品密钥、创建管理员账户、设置终端类型及序号。

### 功能特性
多语言支持：内置 8 种语言（中文、English、日本語、Français、Deutsch、Español、Русский、العربية）

三级权限体系：user / admin / developer（开发者需独立激活密钥）

加密安全：配置文件全程 Fernet 加密，系统密钥 bcrypt 验证，支持暴力锁定

量子终端：超过 80 个内置命令，Tab 补全，历史记录，别名，收藏夹

多会话管理：每个会话拥有独立的虚拟文件系统和真实工作目录

虚拟文件系统：支持创建、读写、删除虚拟文件和目录，支持权限和软链接

定时任务与守护进程：自动备份、定时报告、自愈监控

系统维护：完整性检查、临时文件清理、还原点管理、版本化备份

升级支持：本地版本迁移 + GitHub Release 自动更新

双因素认证：支持 TOTP 动态码保护高权限账户

硬件绑定许可证：可选，防止配置移植

插件系统：支持动态加载 .py 插件扩展终端功能

网络工具：ping、traceroute、安全文件传输（SCP）、邮件通知

沙盒与脚本：.ssea 脚本执行及沙盒模式

自动化测试：完整的 pytest 测试套件

独立 SDK：starsea_sdk 可单独安装，供外部程序调用系统核心 API

官方应用：文件管理器、应用商店、图库、文本编辑器、系统监视器、日历提醒、备份还原向导

### 项目结构
```text
星海量子操作系统/
├── 组件/                     # 系统运行目录
│   ├── main.py               # 主入口
│   ├── requirements.txt      # Python依赖
│   ├── 关于/                 # 产品说明文档
│   ├── 支付码/               # 赞赏码（空白占位）
│   ├── 示例文档/             # 16类空白模板文件
│   ├── 数据集/               # 字符集、表情集、示例表格、测试文本
│   ├── 应用/                 # 官方应用
│   └── system/               # 核心系统模块（40+个Python模块）
├── sdk/                      # 独立 SDK 包
│   └── starsea_sdk/
│       ├── client.py
│       ├── crypto.py
│       └── errors.py
└── G_D_蓝域星河的数据储存库/ # 用户数据仓库（名称可按终端类型定制）
```
### 默认产品密钥
开发测试密钥示例：STAR1-SEA2O-S3QU4-NTUM5-OS123
其 bcrypt 哈希硬编码在 system/config.py 的 VERSION_KEY_HASHES 中，正式部署时请使用密钥生成工具更换。
